# LineNotify4MQL
This is just for giving LINE notification from the indicator side in MT4.

Please use it in other indicators.

It uses WinInet.DLL.

MT4でインジケータ側からLINE通知をだすためだけのものです。

他のインジケータに組み込んでお使いください。

WinInet.DLLを使用します。
